# 📊 Pesquisa Culturama

Projeto simples de um formulário de pesquisa online desenvolvido para a empresa fictícia **Culturama**.

## 🚀 Sobre o projeto

Este projeto consiste em uma página web estática que permite a coleta de informações por meio de um formulário. Após o envio, o usuário é direcionado para uma página de sucesso.

Ideal para estudos de front-end e manipulação básica de formulários.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3

## 📁 Estrutura do projeto

- `index.html` → Página principal com o formulário
- `sucesso.html` → Página exibida após envio
- `/css` → Arquivos de estilo
- `/img` → Imagens utilizadas no projeto
